# Music Player App (Flutter)

## Overview
This project is a feature-rich music player app inspired by Poweramp Pro. Built using **Flutter**, the app supports playing local audio files with essential playback controls and advanced EQ features.

## Features (Initial Phase)
- Local audio playback.
- **Playback Controls:** Play, Pause, Stop, Seek.
- Minimalistic UI for seamless user experience.

## Stack
- Language: Dart
- Framework: Flutter
- Libraries: audioplayers, permission_handler, etc.

## Setup
1. Clone the repo locally.
2. Install dependencies:
    ```bash
    flutter pub get
    ```
3. Run the project:
    ```bash
    flutter run
    ```
